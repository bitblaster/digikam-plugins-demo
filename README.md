* Description:

This repository guest some demo codes to write new external digiKam plugins (Digikam::DPlugin)
as Generic tools, Image Editor tools, and Batch Queue Manager tools.

* Dependencies:

- cmake           >= 3.1.0
- Qt              >= 5.6.0
- DigikamCore     >= 6.3.0
- DigikamDatabase >= 6.3.0
- DigikamGui      >= 6.3.0

* Compilation:

./bootstrap.sh
cd build
make

* Install:

sudo make install/fast

* Uninstall:

sudo make uninstall
